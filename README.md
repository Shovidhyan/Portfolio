If you want to run this project locally on your machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/shovidhyan/Portfolio.git](https://github.com/shovidhyan/Portfolio.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd Portfolio
    ```

    ```bash
    npm install

    ```

    ```bash

    ```
